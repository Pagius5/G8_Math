    二元二次方程组

    关键词

    术语 英文 含义
    二元二次方程组 System of Binary Quadratic Equations 至少含有一个二次方程的二元方程组
    代入消元法 Substitution Method 通过化简其中一个方程，将其代入另一个方程消去未知数
    因式分解法 Factorization Method 将其中一个方程分解为两个一次方程求解
    降次法 Reduction of Degree Method 通过消去常数项或二次项，降低方程次数的求解方法

    二元二次方程组的定义

    定义：二元二次方程组

    一般地，由一个二元二次方程和一个二元一次方程（或两个二元二次方程）组成的方程组，叫做二元二次方程组。

    其一般形式可表示为： \(\begin{cases} f(x, y) = 0 \\ g(x, y) = 0 \end{cases}\)

    其中 \(f(x, y)\) 和 \(g(x, y)\) 至少有一个是关于 \(x, y\) 的二元二次多项式。

    二元一次 + 二元二次方程组

    对于“二元一次 + 二元二次”型的方程组，核心思路是代入消元法，将其转化为一元二次方程求解。

    解法：代入消元法

    步骤：

    1. 整理： 将二元一次方程变形，用含有一个未知数的代数式表示另一个未知数（如 \(y = ax + b\)）。

    2. 代入： 将该表达式代入二元二次方程中，消去一个未知数，得到一个一元二次方程。

    3. 求解： 解该一元二次方程，求出两个未知数的值。

    4. 回代： 将求得的值代入一次方程，求出另一个未知数的值。

    示例： 解方程组 \(\begin{cases} x - y = 1 \\ x^2 + y = 5 \end{cases}\)

    • 由方程1得 \(y = x - 1\)

    • 代入方程2：\(x^2 + (x - 1) = 5 \Rightarrow x^2 + x - 6 = 0\)

    • 解得 \(x_1 = 2, x_2 = -3\)

    • 进而求出 \(y_1 = 1, y_2 = -4\)

    二元二次 + 二元二次方程组

    当两个方程均为二次时，由于形式各异，通常根据方程特点选择因式分解法或降次法。

    1. 因式分解法

    适用于方程可以通过因式分解转化为“一次乘积”形式的情况。根据可分解方程的个数，可以分为以下两种典型情况：

    情况一：仅有一个方程可分解

    如果原方程组中只有一个方程可以分解，即可以化为： $$ \begin{cases} A \cdot B = 0 \\ C = 0 \end{cases} $$ 那么原方程组可以转化为以下两组方程组求解： $$ \begin{cases} A = 0 \\ C = 0 \end{cases} \quad \text{或} \quad \begin{cases} B = 0 \\ C = 0 \end{cases} $$

    情况二：两个方程均可分解

    如果方程组中的两个方程都可化为两个二元一次方程的乘积，那么这个方程组可以转化为四个二元一次方程组求解。

    即：如果原方程组可化为如下形式： $$ \begin{cases} A \cdot B = 0 \\ C \cdot D = 0 \end{cases} $$ 那么原方程组就可以转化为： $$ \begin{cases} A = 0 \\ C = 0 \end{cases} \quad \text{或} \quad \begin{cases} A = 0 \\ D = 0 \end{cases} \quad \text{或} \quad \begin{cases} B = 0 \\ C = 0 \end{cases} \quad \text{或} \quad \begin{cases} B = 0 \\ D = 0 \end{cases} $$ 其基本思想仍是：通过因式分解实现“降次”，从而将复杂的二次问题拆解为多个简单的一次问题。

    总结

    解二元二次方程组的本质是“化归思想”，即通过变形，将高次、多元的方程转化为低次、一元方程，从而利用已有的知识进行求解。
