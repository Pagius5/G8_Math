# 等腰三角形 Isosceles Triangle

### 关键词 Keywords

| 术语     | 英文         | 含义                                   |
| :------- | :----------- | :------------------------------------- |
| 腰       | Leg          | 等腰三角形中相等的两条边               |
| 底边     | Base         | 等腰三角形中不相等的第三条边           |
| 顶角     | Vertex Angle | 两条腰的夹角                           |
| 底角     | Base Angle   | 腰与底边的夹角                         |
| 三线合一 | Three-in-One | 顶角平分线、底边中线、底边上的高线重合 |

---

## 定义与基本性质

!!! note "定义 (Definition)"
    有两条边相等的三角形叫做**等腰三角形**。相等的两边叫做**腰**，另一边叫做**底边**。

!!! theorem "性质：等边对等角"
    等腰三角形的两个底角相等。（简称：**等边对等角**）

!!! theorem "判定：等角对等边"
    如果一个三角形有两个角相等，那么这两个角所对的边也相等。（简称：**等角对等边**）



---

## 核心特性：三线合一

!!! success "三线合一性质 (Three-Lines-in-One)"
    等腰三角形的**顶角平分线**、**底边上的中线**、**底边上的高**互相重合。
    
    在 $\triangle ABC$ 中，若 $AB = AC$：
    
    1. 若 $AD$ 是角平分线，则 $AD \perp BC$ 且 $BD = CD$。
    2. 若 $AD$ 是底边中线，则 $AD \perp BC$ 且 $\angle BAD = \angle CAD$。
    3. 若 $AD$ 是底边上的高，则 $BD = CD$ 且 $\angle BAD = \angle CAD$。




---

## 知识小结与辅助线技巧

!!! info "等边三角形 (Equilateral Triangle)"
    等边三角形是等腰三角形的特殊形式：
    
    - 三边都相等。
    - 三个内角都相等，并且每一个角都等于 $60^\circ$。

??? tip "常见辅助线作法"
    
    1.  **作底边上的高**：利用三线合一性质，将等腰三角形转化为两个全等的直角三角形。
    2.  **作顶角的平分线**：同样可以利用对称性解决角度计算或线段证明问题。
    3.  **连接顶点与底边中点**：这是处理涉及中点问题的首选路径。
