# G8 数学双向细目表

### 说明
评价内容综合参考两个教学体系：**美国 Common Core 大纲**和 **《义务教育数学课程标准 (2022年版)》** 要求，兼顾高中阶段 SAT I/II 和美国 AP 数学的基本能力要求。

**评价水平层级：**

* **1**：了解

* **2**：理解与掌握

* **3**：运用与探究

* **\***：选修

* **0**：未涉及该部分内容
