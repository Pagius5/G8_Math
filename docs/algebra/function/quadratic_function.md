# 二次函数

## 关键词

| 术语     | 英文               | 含义                                              |
| :------- | :----------------- | :------------------------------------------------ |
| 二次函数 | Quadratic Function | 自变量最高次数为 2 的多项式函数                   |
| 抛物线   | Parabola           | 二次函数的几何图像                                |
| 顶点     | Vertex             | 抛物线的最高点或最低点                            |
| 对称轴   | Axis of Symmetry   | 穿过顶点并平分抛物线的垂直直线                    |
| 判别式   | Discriminant       | $\Delta = b^2 - 4ac$，决定图像与 $x$ 轴的交点个数 |

## 二次函数的定义

!!! info "定义：二次函数"
    一般地，形如 $y = ax^2 + bx + c$ ( $a, b, c$ 为常数，且 $a \neq 0$ ) 的函数叫做**二次函数**。

    **注意：**
    
    1. 二次项系数 $a$ 绝对不能为 0。
    2. 定义域为全体实数 $\mathbb{R}$。
    3. 图像是一条**抛物线**。

## 二次函数的三种表示法

根据已知条件（如顶点、截距等），选择合适的表达形式可以极大地简化计算：

| 形式名称   | 代数表达式                | 适用场景                               |
| :--------- | :------------------------ | :------------------------------------- |
| **一般式** | $y = ax^2 + bx + c$       | 基础形式，已知图像上任意三点时使用     |
| **顶点式** | $y = a(x - h)^2 + k$      | 已知**顶点坐标 $(h, k)$** 或最值时使用 |
| **交点式** | $y = a(x - x_1)(x - x_2)$ | 已知图像与 $x$ 轴的**两个交点**时使用  |

!!! theorem "性质：抛物线的形态与位置"
    **1. 开口方向：**
    
    * 当 $a > 0$ 时，抛物线开口**向上**，顶点是最低点。
    * 当 $a < 0$ 时，抛物线开口**向下**，顶点是最高点。

    **2. 顶点与对称轴：**
    对于一般式 $y = ax^2 + bx + c$，其顶点坐标为：
    
    $$\left( -\frac{b}{2a}, \frac{4ac - b^2}{4a} \right)$$

    对称轴方程为：$x = -\frac{b}{2a}$。

    **3. 与 $x$ 轴的交点（根）：**
    通过判别式 $\Delta = b^2 - 4ac$ 判断：
    
    * $\Delta > 0$：图像与 $x$ 轴有 2 个交点。
    * $\Delta = 0$：图像与 $x$ 轴有 1 个交点（顶点在轴上）。
    * $\Delta < 0$：图像与 $x$ 轴无交点（整体在轴上方或下方）。

## 实际应用

二次函数常用于描述具有“极值”特征或受重力影响的过程：

* **物理学（抛体运动）：** 忽略阻力时，物体的飞行轨迹 $h(t) = -\frac{1}{2}gt^2 + v_0t + h_0$ 是一条典型的抛物线。
* **工程学：** 悬索桥的缆绳形状（近似）、卫星天线的反射面（抛物面）设计。
* **经济学：** 总收益或利润函数建模，用于寻找**最大利润**对应的产量。

??? tip "拓展：从配方法到导数"
    在没有微积分的时代，我们通过“配方法”将一般式化为顶点式来寻找最值。但在学习了导数后，我们可以直接通过求导来定位顶点：
    
    设 $f(x) = ax^2 + bx + c$，令其导数 $f'(x) = 2ax + b = 0$，解得 $x = -\frac{b}{2a}$。这就是抛物线顶点的横坐标。这种“变化率为零的点即为极值点”的思想是后续研究复杂曲线的起点。
